Lab 2: HTML


CPIT405

Imagine HTML as the skeleton of a webpage, providing the structure and organization for everything you see. It stands for HyperText Markup Language, and it uses simple tags to define different sections like headings, paragraphs, images, and links. Think of these tags as instructions for the browser, telling it where to place each element and how to display it. While HTML doesn't add fancy colors or interactivity, it's the essential foundation for everything you experience online!

CSS, which stands for Cascading Style Sheets, takes the bare-bones structure of HTML and adds the visual flair. Imagine it as the paint, clothing, and decorations that transform that skeleton into a living, breathing website. It controls the visual presentation of all those HTML elements.

-Font styles and sizes: Want bold headings and elegant body text? CSS makes it happen.


-Colors and backgrounds: Craving a vibrant blue background or a subtle gradient? CSS can paint the picture.


-Layout and positioning: Need elements neatly arranged in columns or a specific image centered on the page? CSS is your layout maestro.


-Decorations and effects: Fancy some borders, shadows, or even animations? CSS adds the pizzazz.


While HTML provides the structure, CSS brings the website to life, making it visually appealing, engaging, and ultimately, a much more pleasant experience for visitors. It's a powerful tool that, paired with HTML, forms the core of web development!

Summary

Think of web development as the process of building the magic behind the websites and web applications you use every day. It encompasses everything from crafting the basic structure and visual styling to adding interactivity, functionality, and security.

Examples

Here's a breakdown of the key aspects:

Example 1

Front-end development: This focuses on the user-facing side, building the visual elements and interactive features that users directly experience. HTML and CSS, as we discussed, are the cornerstones here. JavaScript adds interactivity, allowing users to click, submit forms, and see dynamic content. Frameworks like React or Angular can streamline the process.

Example 2

Back-end development: This is the hidden engine, dealing with data management, server communication, and ensuring smooth operations. Languages like Python or PHP power the server-side, while databases like MySQL store and manage information. Security measures are crucial here to protect user data and website integrity.
